## Viewing run logs

### Suite run logs  
All suite runs—manual, scheduled, or external—are logged.  
1. Go to the suite.  
2. Click the **Logs** tab.  
3. Select a run to see a summary of its tests.

![Scheduling runs](/images/running/suite-logs.png)

### Test run logs  
All test runs are logged, whether run individually or as part of a suite.  
1. Go to the test.  
2. Click the **Logs** tab for a run history.

![Test runs](/images/running/test-page.png)

### Debugging and run proofs  

![Test runs](/images/running/testfailed.png)

- View warnings and errors in the logs.  
- See a GIF of each step.  
- Click **View Trace** for console logs, network logs, and timestamped screenshots.
