### Suite run reports
All suite runs—manual, scheduled, or external—are logged. 
 
![Scheduling runs](/images/running/report.png)
1. Go to the suite.  
2. Click the **Logs** tab.  
3. Select a run to see a report of its tests.
4. Click on any test run to view its logs, GIF, and traces.
![Scheduling runs](/images/running/report-test.png)
