## Scheduling suite runs  
![Scheduling runs](/images/running/schedule.png)
1. Open the suite and select the **Schedules** tab.  
2. Click **Add Schedule**.  
3. Choose hourly or daily recurrence. 
4. Select the config and environment.
5. All schedules for that suite appear in this view.

> **Note:** Tests marked **Draft** will be skipped in suite runs but can be executed individually.
