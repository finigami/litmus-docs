---
title: "Run a Suite from the Dashboard and Suite Page"
description: "A step-by-step guide to running test suites in LitmusCheck from the dashboard and suite page"
---

## Run from dashboard

1. Go to the dashboard
2. Hover over the suite name
3. Click the **Three dots**
4. Click **Run**
![Run test](/images/running/dashboard-run.png)

> **Note:** Tests marked **Draft** will be skipped in suite runs but can be executed individually.

## Run from the suite page (Default Config)

1. Go to the dashboard
2. Open the suite page
3. Click **Run**

## Run from the suite page (Advance Config)

1. Go to the dashboard and open the the suite page
2. Click the dropdown arrow next to **Run**
3. Click **Choose config**
4. Select **Broswer**
5. Select **Device**
6. Select **OS**
7. Select **Viewport**
8. Click **Run Suite**
