## Running suites and tests

- **Suite**: navigate to the suite and click **Run**.  
- **Test**: navigate to the test and click **Run**.
![Run test](/images/running/test-actions.png)

> **Note:** Tests marked **Draft** will be skipped in suite runs but can be executed individually.
